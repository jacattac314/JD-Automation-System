# PWA Icons

Place your PWA icons in this directory. Icons should be PNG format with the following sizes:

## Required Sizes

- `icon-72x72.png` - 72x72 pixels
- `icon-96x96.png` - 96x96 pixels
- `icon-128x128.png` - 128x128 pixels
- `icon-144x144.png` - 144x144 pixels
- `icon-152x152.png` - 152x152 pixels
- `icon-192x192.png` - 192x192 pixels (Android)
- `icon-384x384.png` - 384x384 pixels
- `icon-512x512.png` - 512x512 pixels (Android, maskable)

## Shortcuts (Optional)

- `shortcut-new.png` - 96x96 pixels
- `shortcut-history.png` - 96x96 pixels
- `shortcut-settings.png` - 96x96 pixels

## Auto-Generation

You can use online tools to generate all sizes from a single high-resolution icon:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

The app will work without icons, but they improve the user experience when installed on mobile devices.
